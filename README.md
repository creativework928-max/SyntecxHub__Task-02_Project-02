# 📊 Data Visualization Projects using Python

This repository contains three practical projects focused on data visualization and exploratory analysis using Python. These projects demonstrate how to analyze datasets, uncover insights, and present findings through effective charts and graphs.

---

## 📁 Projects Overview

---

### 🔹 Project 1: Time Series & Category Charts

Analyze and visualize data over time and across categories.

#### 📌 Objectives:

* Plot time-based trends using line charts
* Aggregate data monthly or quarterly
* Compare categories using bar charts
* Show proportions using pie charts

#### 📊 Tasks:

* Plot sales (or similar metric) over time
* Perform time aggregation (monthly/quarterly)
* Create:

  * Line charts (trend analysis)
  * Bar charts (category comparison)
  * Pie charts (distribution/share)
* Save charts as PNG files
* Write a short summary of findings

#### 📈 Key Learnings:

* Time series visualization
* Choosing appropriate chart types
* Formatting labels, legends, and axes

---

### 🔹 Project 2: Statistical Plots & Distribution Analysis

Understand how data is distributed and identify patterns.

#### 📌 Objectives:

* Analyze distributions using statistical plots
* Compare distributions across different groups
* Detect outliers and skewness

#### 📊 Tasks:

* Create:

  * Histograms
  * KDE (Kernel Density Estimation) plots
  * Boxplots
* Compare distributions (e.g., Region A vs Region B)
* Identify:

  * Outliers
  * Skewness
  * Spread of data
* Export plots
* Write a one-paragraph interpretation

#### 📈 Key Learnings:

* Distribution analysis
* Outlier detection
* Understanding skewness and variance

---

### 🔹 Project 3: Correlation Heatmap & Pairwise Relationships

Explore relationships between numerical variables.

#### 📌 Objectives:

* Measure correlation between variables
* Visualize relationships using heatmaps and pairplots

#### 📊 Tasks:

* Compute Pearson correlation coefficients
* Create:

  * Correlation heatmap (with annotations)
  * Pairplot (scatter matrix)
* Highlight:

  * Strong positive correlations
  * Strong negative correlations
* Mask upper triangle in heatmap for clarity
* Summarize key relationships

#### 📈 Key Learnings:

* Feature relationships
* Multivariate analysis
* Identifying important variables

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/creativework928-max/SyntecxHub__All-Projects-02
```

2. Navigate to the folder:

```bash
cd data-visualization-projects
```

3. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

4. Launch Jupyter Notebook:

```bash
jupyter notebook
```
